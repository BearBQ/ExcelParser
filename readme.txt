Программа для парсинга данных из заполненных лотов

Исходный файл необходимо поместить в папку mainfile, переименовать его в main.xlsx

Заполненные лоты необходимо поместить в папку filesExcel, выборка происходит автоматически

В качестве результата - создается файл products_report, а так же копия исходного лота с заполненными значениями в папке Work
